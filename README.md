# ✅ todo-list

💁‍♀️ 바쁘다 바빠! 어디 해야할 일 기록할 곳 없나? <br>
해야할 일을 간편하게 작성하는 todo-list!
<br><br>

## ⚙️ Install
1. clone(SSH) : git@github.com:YouJin-Cho/todo-list.git
2. terminal ➡️ cd todo-list
3. npm install
4. npm run start
<br><br>

## 💪🏻 기술 스택
* TypeScript
* React
* context API
<br><br>
